# Internal Tool Pack

Use this pack for operator dashboards, admin surfaces, and workflow tools.

Priorities:

- fast data entry and editing
- tables, filters, bulk actions, and detail panes
- clear status, error, and audit visibility
- keyboard efficiency over marketing polish

Common concerns:

- role-based visibility
- import and export flows
- empty and loading states for operational data
- safe destructive actions
